-# Training Planner

**Автор:** [Крикунов Алексей]

## Описание
Приложение для записи и отслеживания тренировок. Позволяет добавлять тип занятия, дату и продолжительность, сохраняя данные в локальный JSON-файл.

## Функционал
- Валидация даты и положительного времени.
- Автоматическое сохранение в `trainings.json`.
- Живой поиск (фильтрация) по типу тренировки в таблице.

## Пример использования
1. Запустите `python main.py`.
2. Введите дату `12.05.2024`, тип `Бег`, длительность `45`.
3. Нажмите "Добавить".
4. Чтобы найти все пробежки, введите "бег" в поле фильтра.
